## Overview
In the EPS2 documentation, you will find information about the hardware and software architectures used, as well as tips and procedures for using the board. Additionally, it includes several test reports in the appendices, detailing the tests conducted on different versions of EPS2, along with the results and a brief analysis. This document can be opened in a specific LaTeX editor (e.g., TeXstudio, Texmaker) or on an online platform like Overleaf.
